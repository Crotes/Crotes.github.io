标题标签

基本内容

网页标题分为6个等级，< h1 > - < h6 >

h为head的缩写，意思为头部，标题

语法

标签语义

作为标题使用，依据重要性递减

特点

1.加了标题的文字会加粗，字号变大

2.一个标题独占一行，后面的文字会在下一行显示

段落标签

基本内容

< p >标签用于定义段落，它可以将整个网页分为若干段落

p为paragraph的缩写，意为段落

语法

标签语义

把HTML文档分割为若干段落

特点

1.文本在一个段落中会根据浏览器窗口的大小自动换行

2.段落与段落之间含有空隙

换行标签

基本内容

在HTML中，一个段落中的文字会一直到浏览器右边才会换行，如果要强制换行，需要**换行标签< br / >**，这是一个单标签

b为break的缩写，意为打断

语法

标签语义

强制换行

特点

1.< br / >是一个单标签

2.< br / >标签只是简单地开始新的一行，跟段落不一样，段落之间会插入一些垂直的间距

文本格式化标签

基本内容

在网页中有时需要为文字设置粗体，斜体，下划线等效果，这时需要用到HTML中的文本格式化标签，使文字以特殊方式显示

标签语义

突出重要性，比普通文字更重要

语法

语义 标签 说明
加粗 < strong>< /strong>或< b>< /b> 更推荐使用< strong>标签加粗，语义更强烈
倾斜 < em>< /em>或< i>< /i> 更推荐< em>标签
删除线 < del>< /del>或< s>< /s> 更推荐< del>标签
下划线 < ins>< /ins>或< u>< /u> 更推荐< ins>标签

< div >和< span >标签

基本内容

< div >和< span >没用语义，他们就是一个盒子，用来装内容

div是division的缩写，表示分割，分区

span意为跨度，跨距

语法

特点

1.< div>标签用来布局，但是现在一行只能放一个< div>。

2.< span> 标签用来布局，一行上可以放多个< span>。

图像标签与路径

图像标签

基本内容

< img>标签用于定义HTML页面中的图像

src是< img>标签的必须属性，它用于指定图像文件的路径和文件名

语法

图片属性

属性 属性值 说明
src 图片路径 必须属性
alt 文本 替换文本，图像不能显示的文字
title 文本 提示文本，鼠标放在图像上的文字
width 像素 宽度
height 像素 高度
border 像素 边框粗细

注意点

1.图像标签可以有多个属性，必须写在标签名后面

2.属性之间不分前后，但必须隔开

3.属性采取键值对的格式，key = “value”

路径

路径可以分为：

相对路径：以引用文件所在位置为参考基础，建立出的目录路径

相对路径分类 符号 说明
同一级 图像文件位于html文件的同一级
下一级 / 图像文件位于html文件的下一级
上一级 ../ 图像文件位于html文件的上一级

绝对路径：目录下的绝对位置，直接到达目标，通常从盘开始，注意符号为\

超链接标签

基本内容

< a>标签用于定义超链接，作用是从一个页面链接到另一个页面

分类

1.外部链接 例：< a href= “https://www.baidu.com">百度< /a>

2.内部链接：网站内部页面之间的相互链接，直接链接内部页面名称即可

3.空链接：如果当时并未确定链接目标时，< a href = “#”>首页< /a>

4.下载链接：如果href里面地址是一个文件或压缩包，会下载这个文件

5.网页元素链接：在网页中的各种网页元素，如文本，图像，表格等

6.锚点链接：当我们点击链接，可以快速定位到页面中的某个位置

​ · 在链接文本的href属性中，设置属性值为#名字 的形式，如< a href = “#two”> 第二季 < /a>

​ · 找到目标位置标签，里面添加一个id属性 = 刚才的名字， 如 < h3 id = “two”> 第二季介绍< /h3>

语法

属性

属性 作用
href 用于指定链接目标，当标签有了href时，就具有了超链接功能
target 用于指定链接页面的打开方式，_self为默认值， _ blank 为在新窗口中打开

注释标签

基本内容

添加便于阅读和理解但不需要再页面中的注释文字，就需要使用注释标签

HTML中的注释以“< !–” 开头，以” –>“结束

快捷键 ：ctrl + /

添加注释是为了更好地解释代码的功能，便于相关人员理解和阅读代码

特殊字符

特殊字符 描述 代码
空格 &nbsp
< 小于 &lt
> 大于 &gt
& 和号 &amp
￥ 人民币 &yen
© 版权 &copy
注册商标 &reg
℃ 摄氏度 &deg
± 正负号 &plusmn
× 乘 &times
÷ 除 &divide
² 平方 &sup2
³ 立方 &sup3

表格标签

主要作用

用于显示，展示数据，可以让数据显示非常规整，可读性非常好。

表格不是用来布局页面的，是用来展示数据的。

基本语法

< table> < /table>是用于定义表格的标签

< tr> < /tr>标签用于定义表格中的行，必须嵌套再 < table> < /table>标签中

< td> < /td>用于定义表格中的单元格，必须嵌套在 < tr> < /tr> 中

表头单元格标签

基本内容

一般表头单元格位于表格的第一行或第一列，表头单元格里面的文本内容加粗居中显示

< th> < /th>标签表示HTML表格的表头部分

语法

表格属性

属性名 属性值 描述
align left，center，right 规定表格相对周围元素的对其方式
border 1或“ ” 规定表格单元是否有边框，默认无边框
cellpadding 像素值 规定单元边沿与其内容之间的空白，默认1像素
cellspacing 像素值 规定单元格之间的空白，默认2像素
width 像素值或百分比 规定表格的宽度

表格标签的属性实际开发不常用，通过css设置

如果要用，写在table 里面

表格结构标签

基本内容

< thead>表示表格的头部区域，< tbody>表示表格的主体区域,便于分清表格结构

< thead>内部必须有< th>标签

合并单元格

方式

1.跨行合并：rowspan = “number”

2.跨列合并：colspan = “number”

代码在< td>中编写

目标单元格

跨行：最上侧单元格

跨列：最左侧单元格

步骤

1.确定方式

2.找到目标单元格

3.删除多余单元格

列表标签

主要作用

列表是用来布局的，列表的特点就算整齐，整洁，有序，它会使布局更加自由和方便。

分为三类：无序列表，有序列表，自定义列表

无序列表

主要内容

< ul>标签表示HTML页面中的项目的无序列表，一般会以项目符号呈现列表项，而列表项使用< li>标签定义

基本语法

注意

1.< ul>里面只能放< li>

2.< li>里面可以放任何标签

3.无序列表会带有自己的样式属性，但在实际使用时，我们会使用css

有序列表

主要内容

< ol>标签用于定义有序列表，列表顺序以数字来显示，并且使用< li>标签来定义列表项

基本语法

注意

  1. < ol> < /ol>中只能嵌套< li>< /li>,不能有其他标签或文字
  2. < li> 与< /li>之间相当于一个容器，可以容纳所有元素
  3. 有序列表有自己的样式属性，但实际使用时，用css设置

自定义列表

主要内容

用于对术语化妆名词进行解释和描述

< dl>用于定义描述列表，与< dt>(名字)< dd>(每一个项目的名字)一起使用

基本语法

注意

1.< dl>里面只能有< dt> < dd>

2.< dt>< dd>没有数量限制，可以容纳所有元素

总结

标签名 定义 说明
< ul> < /ul> 无序列表 里面只能有< li>，没有顺序，< li>里面可以有任何标签
< ol> < /ol> 有序列表 里面只能有< li>，有顺序，< li>里面可以放任何标签
< dl>< /dl> 自定义列表 里面只能有< dt> < dd>,< dt> < dd>里面可以放任何标签

表单标签

作用

收集用户信息

组成

表单由表单域，表单控件，提示信息三个部分组成

表单域

表单域是一个包括表单元素的区域

< form>标签用于定义表单域，实现用户信息的收集和传递。

< form>会把他范围内的表单元素信息提交给服务器

基本语法

常用属性

属性 属性值 作用
action url地址 用于指定接受冰数里表单数据的服务器程序的url地址
mathod get/post 设置表单数据的提交方式，取值为get或post
name 名称 指定表单的名称，区分一个页面中的多个表单域

表单控件

作用

定义各种表单元素，表单元素是允许用户在表单中输入或选择的内容控件

< input>表单元素

用于收集用户信息

< input>标签中包含一个type属性，根据不同的type属性值，输入字段拥有很多种形式（文本，复选框，掩码后的文本，单选按钮，按钮）

  1. < input />标签为单标签
  2. type属性设置不同的属性值用来指定不同的控件类型
属性值 描述
botton 定义可点击按钮
checkbox 定义复选框
file 定义输入字段和“浏览”按钮，供文件上传
hidden 定义隐藏的输入字段
image 定义图像形式的提交按钮
password 定义密码字段，该字段中的字符被掩码
radio 定义单选按钮
reset 定义重置按钮，会清除表单中的所有数据
submit 定义提交按钮，会把表单数据发送到服务器
text 定义单行的输入字段，默认宽度为20个字符

3.其他属性

属性 属性值 描述
name 由用户自定义 定义input元素的名称
value 由用户自定义 规定input元素的值
checked checked 规定此input元素首次加载时应被选中
maxlength 正整数 规定输入字段中的字符的最大长度

注意：name和value是每个表单元素都有的属性值，给后台使用；name表单元素的名字，要求单选按钮与复选框要有相同的name值；checked属性主要针对于单元按钮和复选框。

< label>标签

< label>标签为input元素定义标注（标签）

< label>标签用于绑定一个表单元素，当点击< label>标签内的文本时，浏览器就会自动将光标转到或选择对应的表单元素上，用来增加用户体验

< label>标签的for属性应与相关元素的id属性相同

< select>下拉表单元素

多个选项让用户选择

1.< select>里面至少包含一对< option>

2.在< option>中定义selected = “selected”时，当前项为默认选中项
